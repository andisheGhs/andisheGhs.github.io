December 2024 Project

🎯 Interactive Algorithm Visualizer

An interactive web application for visualizing fundamental computer science algorithms. Watch sorting algorithms race against each other, explore graph traversal techniques like DFS and BFS, understand clustering with K-means and Pivot algorithms, and see search algorithms in action. Built with React and TypeScript, featuring real-time animations and synchronized code highlighting to help students and developers understand algorithm behavior step-by-step.

Features: 6 sorting algorithms with race mode, 5 graph algorithms including Dijkstra and Kruskal, multiple clustering visualizations, 6 search algorithms with complexity analysis, and interactive graph building tools.
